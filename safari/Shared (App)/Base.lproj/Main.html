Unable to check if Control Panel for YouTube is enabled in Safari

Control Panel for YouTube is enabled in Safari

To enable Control Panel for YouTube in Safari:

  1. Open the Settings app
  2. Tap Apps → Safari → Extensions
  3. Tap Control Panel for YouTube
  4. Tap Allow Extension
  5. Scroll down to Permissions
  6. Tap m.youtube.com then Allow
  7. Tap youtube.com then Allow

To Enable Control Panel for YouTube in Safari:

  1. Enable Control Panel for YouTube
  2. Click Edit Websites…
  3. Change youtube.com to Allow
